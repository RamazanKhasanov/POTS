
Часть 1. Функция Фибоначчи

5.  a) При N = 40. Рекурсивная реализация функции Фибоначчи,
       без использования хвостовой рекурсии, имеет экспоненциальную
       времененную сложность (каждый вызов функции дважды рекурсивно
       вызывает саму себя). Поэтому при больших значениях сильно увеличивается
       время вычисления из-за ожидающих промежуточных значений.
     
    b) Меньше секунды. При использовании хвостовой рекурсии для
       реализации функции Фибоначчи временная сложность будет линейной, потому
       что рекурсивный вызов вызывается в конце, после всех операций вычислений.
       Это позволяет не ожидать промежуточных значений, без накопления вызовов в стеке.
       Поэтому, вычисление происходит мгновенно.
